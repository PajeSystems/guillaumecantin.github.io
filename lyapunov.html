Lyapunov exponents

Date ven. 01 janvier 2016
Category Bifurcations
Tags Lyapunov / Lorenz / bifurcation / chaos / scipy / python

The Lyapunov exponents are used in the theory of dynamical systems to measure the sensibility to a variation of the initial condition. Positive Lyapunov exponents do not systematically mean chaos. The following programm, written in python , computes the Lyapunov exponents for the Lorenz system, using Wolf, Swinney and Vastano algorithm.

#!/usr/bin/env python3

"""
A programm to compute the Lyapunov exponents of the Lorenz system
"""

# Scientific libraries

import numpy as np
from scipy.integrate import odeint
from math import log as ln
from math import sqrt

# Inner product and norm

def ps(u,v):
  u1, u2, u3 = u
  v1, v2, v3 = v
  return float(u1*v1 + u2*v2 + u3*v3)

def norme(u):
  return sqrt(ps(u,u))

# Lorenz system

# Parameters
s, r, b = 16, 45.92, 4

# Lorenz equations
def l(X, t):
  x, y, z = X
  dx = s*(y-x)
  dy = x*(r-z)-y
  dz = x*y-b*z
  return [dx, dy, dz]

# Extended system
def L(V, t):
  x, y, z, v11, v12, v13, v21, v22, v23, v31, v32, v33 = V
  dx = s*(y-x)
  dy = x*(r-z)-y
  dz = x*y-b*z
  dv11 = -s * v11 + s * v12
  dv12 = (r-z)*v11 - v12 -x*v13
  dv13 = y*v11 + x*v12 - b*v13
  dv21 = -s * v21 + s * v22
  dv22 = (r-z)*v21 - v22 -x*v23
  dv23 = y*v21 + x*v22 - b*v23
  dv31 = -s * v31 + s * v32
  dv32 = (r-z)*v31 - v32 -x*v33
  dv33 = y*v31 + x*v32 - b*v33
  return [dx, dy, dz, dv11, dv12, dv13, dv21, dv22, dv23, dv31, dv32, dv33]

X0 = [10, 1, 0]
dt = 0.001
T = 30
N = int(T/dt)

# Integrate until attractor
orbit = odeint(l, X0, np.arange(0, 5, dt))
X0 = orbit[-1]

print('Beginning calculating',end='')

v1 = np.matrix([1, 0, 0]).T
v2 = np.matrix([0, 1, 0]).T
v3 = np.matrix([0, 0, 1]).T

# Compute an orbit
orbit = odeint(l, X0, np.arange(0, T, dt))

LE1 = 0
LE2 = 0
LE3 = 0

for k in range(N):
  if k%100 == 0:
    print('.',end='')
  x, y, z = orbit[k][0], orbit[k][1], orbit[k][2]
  V0 = x, y, z, float(v1[0]), float(v1[1]), float(v1[2]), float(v2[0]), float(v2[1]), float(v2[2]),float(v3[0]), float(v3[1]), float(v3[2])

  # Integration for one step
  V = odeint(L, V0, [0, dt])
  v1 = np.matrix([V[-1][3], V[-1][4], V[-1][5]]).T
  v2 = np.matrix([V[-1][6], V[-1][7], V[-1][8]]).T
  v3 = np.matrix([V[-1][9], V[-1][10], V[-1][11]]).T

  # Accumulation of the deviation
  LE1 = LE1 + ln(norme(v1))
  LE2 = LE2 + ln(norme(v2))
  LE3 = LE3 + ln(norme(v3))

  # Gram-Schmidt method
  v1 = v1/norme(v1)
  v2 = (v2 - ps(v2,v1)*v1)/norme(v2 - ps(v2,v1)*v1)
  v3 = (v3 - ps(v3,v2)*v2 - ps(v3,v1)*v1)/norme(v3 - ps(v3,v2)*v2 - ps(v3,v1)*v1)

print('\n')

LE1, LE2, LE3 = LE1/T, LE2/T, LE3/T

print('Lyapunov exponents :',LE1, LE2, LE3)
